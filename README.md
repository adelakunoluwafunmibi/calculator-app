# Calculator App

A scientific calculator built with React + TypeScript + Vite, made for the SEN 104 / SEN 214 mobile app assignment.

## Features

**Basic mode**
- Addition, subtraction, multiplication, division
- Percent, backspace, clear

**Scientific mode**
- Trigonometric functions: sin, cos, tan
- Hyperbolic functions: sinh, cosh, tanh
- log, ln, square root, cube root, square, power (xʸ)
- Factorial (x!), permutations (nPr), combinations (nCr)
- Constants: π, e
- 1/x, |x|, ±

## Running locally

```bash
npm install
npm run dev
```
